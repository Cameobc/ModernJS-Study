07장 - 연산자 연산자의 종류 : 산술, 할당, 비교, 논리, 타입, 지수 연산 연산의 대상 : 피연산자 ( 값으로 표현될 수 있는 표현식 ) 연산자 : 피연산자를 연산하여 새로운 값을 만드는 역할 1. 산술 연산자 - 산술 연산자로 새로운 값을 만들며 불가능한경우 NaN을 반환 1-1 이항 산술 연산자로 - 2개의 피연산자 사용, 피연산자의 값이 바뀌지 않고 언제나 새로운 값을 만듦 종류 : +, -, *, /, % 1-2 단항 산술 연산자 - 1개의 피연산자 사용 종류 : ++, --, +, -( 양수를 음수로, 음수를 양수로 반전한 값 반환 ) - 암묵적 할당 1-3 문자열 연결 연산자 - (+)연산자는 피연산자 중 하나 이상이 문자열인 경우 문자열 연결 연산자로 동작 example) '1' + 2 //'12' 1+ '2' //'12' 1 + true // 2 암묵적 타입변환, 타입 강제 변환 1 + false // 1 1 + null // 1 +undefined // NaN 1 + undefined // NaN 2. 할당 연산자 - 할당 연산자는 좌항에 변수 값을 할당 - 할당문은 표현식인 문으로, 할당된 값으로 평가 --> 여러 변수에 동일한 값을 연쇄 할당 가능 3. 비교 연산자 - 동등(==)/일치(===) 비교 연산자 - 동등 연산자는 암묵적 타입 변환을 통해 값만 비교 - 일치 연산자는 타입과 값 모두 비교 - NaN은 자신과 일치하지 않는 유일한 값 --> NaN===NaN; //false - Object.is 는 NaN비교 true값 반환 나머지는 일치 비교 연산자와 동일 - 대소 관계 비교 연산자 <,>,<=,>= - 삼항 조건 연산자 : 값으로 평가할 수 있는 표현식인 문 examle) var x = 2; var result = x % 2 ? '홀수' : '짝수'; 조건 ? '참' : '거짓' 반환 4. 논리 연산자종류 종류 : ||(논리합), &&(논리곱), !(부정) !'Hi'; // false 'Cat' && 'Dog'; // 'Dog' 5. typeof 연산자 - 피연산자의 데이터 타입을 문자열로 반환, 값이 null타입인지 확인하고 싶을 때는 (===)일치 연산자 사용 - 선언하지 않은 식별자를 typeof으로 연산시 참조에러발생x 고 undefined 반환 6. 지수 연산자 - ** 로 사용, 이전엔 Math.pow사용 - 음수 제곱시 (-5) ** 2 = 25처럼 괄호로 묶기 - 할당문과 같이 사용가능 (num **= 2) *. 그 외의 연산자 - ?. : 옵셔널 체이닝 연산자 - ?? : null 병합 연산자 - delete : 프로퍼티 삭제 - new : 생성자 함수를 호출할 때 사용하여 인스턴스 생성 - instanceof : 좌변의 객체가 우변의 생성자 함수와 연결된 인스턴스인지 판별 - in : 프로퍼티 존재 확인 *. 연산자 우선순위 - ()를 사용해 명시적으로 우선순위 조절 권장
